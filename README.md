# PD Portfolio VS Code Kit

이 폴더를 Visual Studio Code로 열면 바로 수정할 수 있습니다.

- `index.html`: 문구와 섹션 구조
- `styles.css`: 색감, 폰트, 반응형 디자인
- `assets/hero-prism-studio.png`: 직접 생성한 히어로 비주얼

수정 추천 순서:

1. `지원자 이름`, 이메일, 전화번호를 실제 정보로 교체
2. `CASE DECK`의 샘플 프로젝트를 실제 영상/기획서/성과로 교체
3. 다른 회사에 지원할 때는 `TARGET FIT` 섹션의 회사명과 세 가지 핏만 교체
4. 브라우저에서 `index.html`을 열어 확인
